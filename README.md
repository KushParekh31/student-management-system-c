# Student Management System in C

A simple console-based Student Management System developed in C. This project allows users to add, view, and search student records using structures, arrays, and functions.

## Features

* Add Student Records
* Remove Student Records
* Update Student Records
* Display All Students
* Menu-Driven Interface

## Technologies Used

* C Programming Language
* Structures
* Arrays
* Functions
* Loops and Conditional Statements

## Project Structure

```text
StudentManagementSystem/
│
├── main.c
└── README.md
```

## Sample Menu

```text
===== STUDENT MANAGEMENT SYSTEM =====
1. Add New Student's Data
2. Remove Selected Student's Data
3. Update Student's Data
4. Display Student's Data
5. Exit Program
```

## Student Information

Each student record contains:

* Student ID
* Student Name
* Marks

## How to Run

Compile the program:

```bash
gcc main.c -o sms
```

Run the executable:

```bash
./sms
```
